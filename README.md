# Μεζεδοπωλείο Management System

## Περιγραφή
Διαδικτυακή εφαρμογή διαχείρισης μεζεδοπωλείου που αναπτύχθηκε με PHP, MySQL και XAMPP.

Η εφαρμογή υποστηρίζει:
- διαχείριση προϊόντων
- διαχείριση προμηθευτών
- παραγγελίες προμηθευτών
- έλεγχο χαμηλού αποθέματος

---

## Τεχνολογίες
- PHP
- MySQL / MariaDB
- Apache (XAMPP)
- HTML
- CSS
- VSCode

---

## Λειτουργίες Συστήματος

### Προϊόντα
- Προσθήκη προϊόντων
- Προβολή αποθήκης
- Διαγραφή προϊόντων

### Προμηθευτές
- Καταχώρηση προμηθευτών
- Επιλογή προμηθευτή από dropdown menus

### Παραγγελίες
- Καταχώρηση παραγγελιών προς προμηθευτές
- Προβολή παραγγελιών

### Low Stock
- Αυτόματη εμφάνιση προϊόντων με χαμηλό απόθεμα

---

## Εγκατάσταση

1. Εγκατάσταση XAMPP
2. Εκκίνηση Apache και MySQL
3. Αντιγραφή φακέλου στο:
C:\xampp\htdocs\

4. Import του database.sql μέσω phpMyAdmin
5. Άνοιγμα:
http://localhost/mezedopoleio
