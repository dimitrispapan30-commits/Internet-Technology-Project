# Μεζεδοπωλείο Management System

## 1. Περιγραφή Σεναρίου

Η εφαρμογή αναπτύχθηκε για τη διαχείριση ενός μεζεδοπωλείου. Η επιχείρηση συνεργάζεται με πολλούς προμηθευτές για την προμήθεια πρώτων υλών και προϊόντων.

Το σύστημα επιτρέπει:

* Καταχώρηση προϊόντων στην αποθήκη.
* Καταχώρηση και διαχείριση προμηθευτών.
* Καταχώρηση παραγγελιών προς προμηθευτές.
* Παρακολούθηση αποθεμάτων.
* Αυτόματο εντοπισμό προϊόντων με χαμηλό απόθεμα.

Στόχος της εφαρμογής είναι η ψηφιακή διαχείριση της αποθήκης και της εφοδιαστικής αλυσίδας του μεζεδοπωλείου.

---

## 2. Αρχιτεκτονική Εφαρμογής

Η εφαρμογή ακολουθεί αρχιτεκτονική Client – Server.

### Frontend

* HTML
* CSS

### Backend

* PHP

### Database

* MySQL / MariaDB

### Server

* Apache μέσω XAMPP

Ο χρήστης αλληλεπιδρά μέσω browser, η PHP επεξεργάζεται τα δεδομένα και η βάση MySQL αποθηκεύει τις πληροφορίες.

## 3. Δομή Βάσης Δεδομένων

Η βάση δεδομένων mezedopoleio_db αποτελείται από τους εξής πίνακες:

### products

Αποθηκεύει τα προϊόντα.

Πεδία:

* id
* name
* category
* quantity
* supplier
* unit

### suppliers

Αποθηκεύει τους προμηθευτές.

Πεδία:

* id
* name
* phone
* category

### orders

Αποθηκεύει τις παραγγελίες.

Πεδία:

* id
* product
* quantity
* supplier

## 4. Διαδικασία Ανάπτυξης

Η ανάπτυξη πραγματοποιήθηκε με τα εξής βήματα:

1. Εγκατάσταση XAMPP.

3. Δημιουργία των πινάκων products, suppliers και orders.
4. Υλοποίηση σύνδεσης PHP – MySQL.
5. Δημιουργία σελίδας καταχώρησης προϊόντων.
6. Δημιουργία σελίδας διαχείρισης προμηθευτών.
7. Δημιουργία σελίδας παραγγελιών.
8. Δημιουργία σελίδας προβολής προϊόντων.
9. Υλοποίηση λειτουργίας χαμηλού αποθέματος.
10. Βελτίωση διεπαφής χρήστη με CSS.
11. Δημοσίευση κώδικα στο GitHub.

## 5. Δικτυακή Αρχιτεκτονική

Η δικτυακή προσομοίωση υλοποιήθηκε στο Cisco Packet Tracer.

Η τοπολογία περιλαμβάνει:

* 5 Routers
* 2 Switches
* 2 PCs
* 1 Web Server

Χρησιμοποιήθηκε κυκλική (Ring) τοπολογία ώστε να υπάρχει εναλλακτική διαδρομή επικοινωνίας σε περίπτωση αστοχίας συνδέσμου.

Ο Web Server φιλοξενεί HTML έκδοση της εφαρμογής για την επίδειξη πρόσβασης μέσω HTTP.

Πραγματοποιήθηκαν:

* Ping tests
* Έλεγχος δρομολόγησης
* Πρόσβαση μέσω HTTP Browser

## 7. Εγκατάσταση

1. Εγκατάσταση XAMPP.
2. Εκκίνηση Apache και MySQL.
3. Αντιγραφή φακέλου στο:

C:\xampp\htdocs\Mezedopoleio

4. Import του αρχείου database.sql μέσω phpMyAdmin.
5. Άνοιγμα:

http://localhost/Mezedopoleio
